# cpu-control-c

Thermal based CPU Control in C for Linux.

## Install / {Un,Re}install

### Binary

```sh
make install
```

### Binary and autostart with SystemD, systemwise

```sh
sudo ./systemd-install
```

```sh
sudo ./systemd-uninstall
```

```sh
sudo ./systemd-reinstall
```
